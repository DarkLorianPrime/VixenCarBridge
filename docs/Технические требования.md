### Техническое описание проекта
Разработка веб-приложения (Frontend, backend, CI/CD).

#### Требуемые технологии:
##### Frontend:
- React
- Craco
- React-hook-forms
- React-router-dom

##### Backend:
- FastAPI
- SQLModel (Optional: SQLAlchemy)
- Poetry
- Alembic (Migrations)
- Httpx (Optional: Requests)
- AsyncPG
- MinIO
- Uvicorn
- RabbitMQ
- Celery
- Aiogram
- Swagger\\ReDoc (OpenAPI 3.0)
- Logging

##### CI/CD 
- Docker-compose
- Actions
- Nginx

##### Other
- Bash (Config generator)
